# 🎰 Bingo Digital

Sistema de bingo digital para eventos da **Fraternidade Sem Fronteiras**.

## 📋 Sobre

Plataforma web para gerenciar e jogar bingo em tempo real com:
- Painel de administração para sorteios
- Cartelas dos jogadores com marcação de números
- Versão idoso com botões maiores
- Realtime com Supabase
- Suporte a múltiplos eventos simultâneos

## 🚀 Stack

- **Frontend**: React 18.3 + TypeScript + Next.js 15
- **Estilo**: Tailwind CSS + shadcn/ui
- **Backend**: Supabase (Auth + PostgreSQL + Realtime)
- **Deploy**: Vercel
- **Testes**: Vitest + Playwright

## 📦 Instalação

```bash
# Clonar o repositório
git clone https://github.com/seu-usuario/bingo-digital.git
cd bingo-digital

# Instalar dependências
npm install

# Configurar variáveis de ambiente
cp .env.example .env.local

# Rodar em desenvolvimento
npm run dev
```

Acesse: [http://localhost:3000](http://localhost:3000)

## 🔧 Comandos

```bash
npm run dev      # Desenvolvimento
npm run build    # Build produção
npm run start    # Rodar produção
npm run test     # Testes Vitest
npm run e2e      # Testes Playwright
npm run lint     # ESLint
```

## 🎯 Funcionalidades

- ✅ Login seguro com Supabase Auth
- ✅ Sorteio de números em tempo real
- ✅ Cartelas com marcação automática
- ✅ Validação de BINGO
- ✅ Histórico de ganhadores
- ✅ QR Code para acesso rápido
- ✅ Versão acessível para idosos
